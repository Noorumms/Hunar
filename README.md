# Hunar — Hyperlocal Skilled Worker Marketplace

> "Har Kaam Ka Ustaad" — Master of Every Job

A full-stack web application connecting customers with nearby skilled 
workers (electricians, plumbers, tutors, AC repair, and more) across 
Pakistani cities.

## Tech Stack

- **Backend:** ASP.NET Core Web API (.NET 8)
- **Architecture:** Clean Architecture (Domain, Application, Infrastructure, API)
- **Database:** SQL Server + Entity Framework Core (Code First)
- **Auth:** JWT Authentication with Role-based Access
- **Frontend:** HTML + CSS + Vanilla JavaScript

## Features

- Browse and search service providers by category and location
- JWT secured authentication with Customer, Provider, and Admin roles
- Full booking workflow — Requested → Confirmed → Completed → Cancelled
- Provider profiles with service listings and pricing
- Reviews and ratings system
- Admin moderation dashboard
- Request logging and global error handling middleware
- AI-powered provider matching (coming soon)
